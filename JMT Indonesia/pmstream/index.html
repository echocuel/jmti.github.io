Start a live video streaming system with just mobile device and cloud

Live video streaming can immidiately be started with just mobile device and network connectivity to shoot and view the video.

Video is streamed appropriately for each device in real time using our own adaptive streaming technology.

The server checks the bandwidth of both shooting end and viewing end in real time and adapts the framerate and resolution of video data to be streamed. This technology allows stable streaming of highest possible quality despite bandwidth instability.
