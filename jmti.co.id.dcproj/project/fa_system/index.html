FAソフトウェアの設計・開発

製造工場向けのソフトウェアの開発を行います。
製造工場ではオートメーション化が進み、単に自動化だけでなく採取したデータから診断や解析を行い独自のノウハウを蓄え、ラインの故障停止を最低限に防ぐことで生産に寄与する試みが行われています。

数々のシステム開発を通じて蓄えたノウハウをベースに、品質第一を心がけ、お客様のご要望に応じたソフトウェアの開発を行います。

  • Windows NT/XP
  • 開発言語 … VB, C, C#
SEMIソフトウェアの設計・開発

SECS（SEMI規格）を使用した、半導体製造装置との通信制御に関連するソフトウェアの開発を行います。

お客様のホストコンピュータに対応した通信仕様で半導体製造装置との通信制御を行うと共に、各種ロボット制御を行い、ロットオペレーションのオートメーション化を実現いたします。

  • Windows 98/NT/2000/XP
  • 開発言語 … VB, C, C#

開発例

半導体製造装置とデバイスメーカーのHOSTコンピュータをオンライン化し、ロット制御のオートメーション化を実現しています。
